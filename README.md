## Description

- This React application is built using [create-react-app](https://create-react-app.dev/).
- It uses [Tailwind CSS](https://tailwindcss.com/)

## Running in dev environment

1. `cd YOUR_PROJECT_NAME`
2. `npm install`
3. `npm start`
